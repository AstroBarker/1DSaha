## Building
To build `SimpleSaha`, simply
```sh
mkdir build && cd build
cmake ..
make -j
```
Then an executable `saha` will be created in `build/src`.

## Clang-format

We use clang format for code cleanliness. 
The current version of `clang-format` used is `clang-format-13`.
